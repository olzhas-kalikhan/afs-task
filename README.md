# AFS APP 

[afs-task app link](https://git.olzhas.dev/afs-task/)

stack: 
- typescript
- react
- sass
- mobx
- react-hook-form
- react-query
- axios
- zod
- radix ui (checkbox,dialog,popover,select)


### How to start locally

`npm install`
`npm run dev`

### How to preview prod

`npm install`
`npm build`
`npm run preview`
